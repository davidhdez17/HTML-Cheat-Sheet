David's Cheat Sheet

Text HTML Tags
Tag Name Description
<h1> Header There are 6 header tags that can be used. h1 thu h6.
<p> Paragraph The paragrpah tag is used to display basic text at its most basic for.
<a> Anchor Used to create a hyperlink
<strong> Strong The strong tag will display bold text.



Table HTML Tags
Tag Name Description
<table> table The tag used to start a table. It is composed of a "head" "body" and "foot"
<thead> Table Head This is used to identify the colums or rows.
<tbody> Table Body Used to identify where the data is located
<tr> Table Row Used to create row for in the table.
<td> Table Data Holds the data displayed in each column.
<tfoot> Table Foot Defines the lowest set of rows in the table.



Table HTML Attributes
Attribute Name Description
align Align Aligns the table to text on the page.
colspan Column Span Defines how many columns the td element should span
rowspan Row Span Defines how many rows the td element should span
colspan Column Span Defines how many columns the td element should span



List HTML Tags
Tag Name Description
<list> List Wrapper element for lists.
<ul> Un Ordered List The type of list created. Displays bullet points.
<ol> Ordered List The type of list created. Displays numbers in order.
